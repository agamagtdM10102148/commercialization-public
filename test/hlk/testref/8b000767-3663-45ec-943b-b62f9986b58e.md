# <span id="p_hlk_test.8b000767-3663-45ec-943b-b62f9986b58e"></span>MTP Compliance Test - Core - Service Fundamentals


This test validates compliance with the Windows implementation of the Media Transfer Protocol (MTP) and ensures that devices use the MTP class driver that is included with Windows.

## Test details

|||
|---|---|
| **Specifications**  | <ul><li>Device.Portable.Core.MTP</li><li>Device.Portable.Core.CustomDeviceServices</li></ul> |  
| **Platforms**   | <ul><li>Windows 10, client editions (x86)</li><li>Windows 10, client editions (x64)</li></ul> |
| **Supported Releases** | <ul><li>Windows 10</li><li>Windows 10, version 1511</li><li>Windows 10, version 1607</li><li>Windows 10, version 1703</li><li>Windows 10, version 1709</li><li>Windows 10, version 1803</li><li>Next update to Windows 10</li></ul> |
|**Expected run time (in minutes)**| 5 |
|**Category**| Development |
|**Timeout (in minutes)**| 300 |
|**Requires reboot**| false |
|**Requires special configuration**| false |
|**Type**| automatic |
