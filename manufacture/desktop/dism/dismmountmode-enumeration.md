# DismMountMode Enumeration


Specifies whether an image is mounted as read-only or as read-write.

## <span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>Syntax


``` syntax
enum DismMountMode
{
    DismReadWrite = 0,
    DismReadOnly = 1
};
```
