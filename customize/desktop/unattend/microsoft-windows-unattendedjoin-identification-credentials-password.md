# Password


`Password` specifies the password of the user account used for authentication of an account to the domain before the computer can be joined to a domain during WindowsÂ Setup.

## XML Example


The following XML output shows how to set the identification credentials.

```
<Identification>
   <Credentials>
      <Domain>fabrikam.com</Domain>
      <Password>MyPassword</Password>
      <Username>MyUserName</Username>
   </Credentials>
   <JoinDomain>fabrikam.com</JoinDomain>
   <MachinePassword>ComputerPassword</MachinePassword>
</Identification>
```
