# NotInOOBE


`NotInOOBE` hides mobile broadband devices or networks in OOBE. You can use this setting if you have a mobile device with embedded mobile broadband thatâ€™s sold with a deactivated mobile operator SIM. In this case, the mobile operator requires the mobile broadband device to be hidden in OOBE because SIM activation requires use of a mobile operator account experience app or a web browser, which are not supported in OOBE.

The following XML output shows how to hide the mobile broadband category in OOBE.

```
<NotInOOBE>true</NotInOOBE>
```
