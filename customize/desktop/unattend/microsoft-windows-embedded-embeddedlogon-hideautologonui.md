# HideAutoLogonUI


Hides the Welcome screen when automatic sign-in is enabled.

By default, custom logon launches directly into the shell without displaying the sign-in UI when automatic sign-in is enabled. You can disable this option to mirror a WindowsÂ 10 experience, which briefly shows the sign-in UI before switching to the shell.
