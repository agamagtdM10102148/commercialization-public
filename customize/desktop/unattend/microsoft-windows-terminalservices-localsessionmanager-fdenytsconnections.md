# fDenyTSConnections


`fDenyTSConnections` specifies whether Remote Desktop connections are enabled.

There are several settings that must be configured to enable Remote Desktop connections during an unattended installation. First, you must enable Remote Desktop connections by using the `fDenyTSConnections` setting. You must also enable the Remote Desktop group in Windows Firewall. For more information, see "Enable Remote Desktop" in the Windows Assessment and Deployment Kit (WindowsÂ ADK).
