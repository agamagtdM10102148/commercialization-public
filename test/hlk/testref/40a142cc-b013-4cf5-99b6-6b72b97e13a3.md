# NDISTest 6.5 - \[1 Machine\] - VMQTestingOIDs


This test validates the VMQ functionality of the miniport adapter. VMQ functionality is optional so the test will successfully exit if the miniport adapter does not support VMQ.

## Test details

|||
|---|---|
| **Specifications**  | <ul><li>Device.Network.LAN.VMQ.VirtualMachineQueues</li></ul> |  
| **Platforms**   | <ul><li>Windows 10, client editions (x64)</li><li>Windows Server 2016 (x64)</li></ul> |
| **Supported Releases** | <ul><li>Windows 10</li><li>Windows 10, version 1511</li><li>Windows 10, version 1607</li><li>Windows 10, version 1703</li><li>Windows 10, version 1709</li><li>Windows 10, version 1803</li><li>Next update to Windows 10</li></ul> |
|**Expected run time (in minutes)**| 5 |
|**Category**| Development |
|**Timeout (in minutes)**| 300 |
|**Requires reboot**| false |
|**Requires special configuration**| false |
|**Type**| automatic |
