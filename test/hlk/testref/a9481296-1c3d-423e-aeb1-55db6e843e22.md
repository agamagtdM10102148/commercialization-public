# <span id="p_hlk_test.a9481296-1c3d-423e-aeb1-55db6e843e22"></span>Trim Zero-Range Test


This test verifies hard disk trim functionality if the device includes ATA non-NCQ Trim support.

## Test details

|||
|---|---|
| **Specifications**  | <ul><li>Device.Storage.Hd.ReadZeroOnTrimUnmap.BasicFunction</li></ul> |  
| **Platforms**   | <ul><li>Windows 10, client editions (x86)</li><li>Windows 10, client editions (x64)</li><li>Windows Server 2016 (x64)</li></ul> |
| **Supported Releases** | <ul><li>Windows 10</li><li>Windows 10, version 1511</li><li>Windows 10, version 1607</li><li>Windows 10, version 1703</li><li>Windows 10, version 1709</li><li>Windows 10, version 1803</li><li>Next update to Windows 10</li></ul> |
|**Expected run time (in minutes)**| 5 |
|**Category**| Compatibility |
|**Timeout (in minutes)**| 300 |
|**Requires reboot**| false |
|**Requires special configuration**| false |
|**Type**| automatic |
