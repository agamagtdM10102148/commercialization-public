# <span id="p_hlk_test.0dd672ba-55d9-4138-9b9a-129db683506f"></span>Wlan Test - Check Hosted Network Capable


This automated test queries whether the device is "Hosted Network" capable.

## Test details

|||
|---|---|
| **Specifications**  | <ul><li>Device.Network.WLAN.SupportHostedNetwork.HostedNetwork</li></ul> |  
| **Platforms**   | <ul><li>Windows 10, client editions (x86)</li><li>Windows 10, client editions (x64)</li><li>Windows 10, mobile edition (ARM)</li></ul> |
| **Supported Releases** | <ul><li>Windows 10</li><li>Windows 10, version 1511</li><li>Windows 10, version 1607</li><li>Windows 10, version 1703</li><li>Windows 10, version 1709</li><li>Windows 10, version 1803</li><li>Next update to Windows 10</li></ul> |
|**Expected run time (in minutes)**| 1 |
|**Category**| Development |
|**Timeout (in minutes)**| 5 |
|**Requires reboot**| false |
|**Requires special configuration**| true |
|**Type**| automatic |

## <span id="Running_the_test"></span><span id="running_the_test"></span><span id="RUNNING_THE_TEST"></span>Running the test


Before you run the test, complete the test setup as described in the test requirements: [Wireless LAN (802.11) Testing Prerequisites](wireless-lan--80211--testing-prerequisites.md).

The test will check whether the device and driver is hosted network capable. The test will fail if the device or driver reports as not hosted network capable.
