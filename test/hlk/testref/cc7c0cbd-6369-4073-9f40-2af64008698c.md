# <span id="p_hlk_test.cc7c0cbd-6369-4073-9f40-2af64008698c"></span>PGremlin Rendering Test


Pgremlin tests DDI calls that a driver can make. This test produces numerous pages of output with shapes and gradient fills / alpha blends. It also prints and tests some complex fonts.
