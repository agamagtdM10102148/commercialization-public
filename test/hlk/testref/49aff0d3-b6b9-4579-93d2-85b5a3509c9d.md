# <span id="p_hlk_test.49aff0d3-b6b9-4579-93d2-85b5a3509c9d"></span>Present Validation 2 - Vista


This automated test verifies the new **D3DPRESENT\_DONOTFLIP** option. The driver should skip the **Present()** method if this option is set.

The test verifies that no presentation occurs when **D3DPRESENT\_DONOTFLIP** is specified. The test first renders with regular **Present()** and then verifies that the front buffer is not modified when the option is used.

This topic applies to the following test jobs:

-   Present Validation 2 - Vista

-   Present Validation 2 - Vista (WoW64)
