# DismPackageIdentifier Enumeration


Specifies whether a package is identified by name or by file path.

## <span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>Syntax


``` syntax
enum DismPackageIdentifier
{
    DismPackageNone = 0,
    DismPackageName = 1,
    DismPackagePath = 2
};
```
