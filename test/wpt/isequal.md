# IsEqual


Compares two [IProfile](iprofile.md) objects.

## Syntax


```
HRESULT IsEqual
  ([in] IProfile* pProfile)
;
```

## Parameters


<a href="" id="pprofile"></a>*pProfile*  
\[in\] A pointer to an **IProfile** object to be compared.
