# DismString structure


DISM API functions that return strings wrap the heap allocated **PCWSTR** in a DismString structure.

Syntax
---

```
typedef struct _DismString {
  PCWSTR Value;
} DismString;
```

Members
----

**Value**  
A null-terminated Unicode string.
